## Getting Started

### Installation

1. Clone the repo
   ```sh
   git@github.com:renatozr/alpinejs-exercises.git
   ```
2. Enter the directory
   ```sh
   cd alpinejs-exercises
   ```
3. Run the project with Live Server or open the `index.html` file with your browser
